    Python Projects

    Brian's Xbox IP Resolver

    Brian's Xbox IP Resolver (Gamertag2IP) is a script created in Python that allows you to search user created database for Xbox users gamertags, IP addresses, XUIDs, and Machine IDs with just their gamertag. The script comes with a main database that is updated often as well.

    IOS Repo Utilities

    IOS Repo Utilities provides a few tools to help IOS jailbreak developers who run a repo or for people looking to create a repo. You can also generate a "Packages" and "control" files and more.

    TheTechGame Video Uploader

    This is a script created in Python with Selenium that downloads videos from YouTube.com and then uploads them to TheTechGame.com.

    YouTube Song Downloader

    Song downloader made in Python. Download a song from YouTube in mp3 format for your phone, class, or anything else. This script needs Chromedriver to run.

    Sileo Native Depiction Generator
